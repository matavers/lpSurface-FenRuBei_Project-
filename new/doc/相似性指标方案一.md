相似性判定模块设计
1. 基本定义
对于网格 M=(V,E,F)，每个顶点 v 已知：
	法向量 n(v)
	主曲率 \kappa_1(v),\kappa_2(v)，满足 \mid\kappa_1\mid\geq\mid\kappa_2\mid
	主方向 d_1(v),d_2(v)（单位切向量，分别对应 \kappa_1,\kappa_2）
	高斯曲率 K(v)，平均曲率 H(v)
	直纹面逼近误差 e_{\mathrm{rolled}}(v)
此外，每个顶点存储其局部度量张量 M(v)，定义如下：
M(v)=I+\alpha\left(\mid\kappa_1(v)\mid\mathrm{\thinsp} d_1(v)d_1(v)^T+\mid\kappa_2(v)\mid\mathrm{\thinsp} d_2(v)d_2(v)^T\right)

其中 \alpha\geq0 是一个全局超参数，控制曲率对距离拉伸的强度。当 \alpha=0 时退化为欧氏度量。
	若某方向曲率绝对值大，则在该方向上的局部距离被拉伸（有效距离变大），从而分区不易沿该方向生长。
	对于圆柱：环向曲率大，轴向曲率为0，因此分区优先沿轴向生长，形成长条形。
	对于圆锥：母线方向曲率小（接近于0），环向曲率较大，分区沿母线方向生长，但由于母线方向逐渐变化，分区形状自然呈三角形。
	对于平面或扭曲平面：曲率处处很小，度量接近各向同性，分区近似圆形（或由测地距离决定的形状）。
2. 各向异性测地距离计算
给定基准点 b，定义从 b 到任意顶点 v 的各向异性测地距离 D(b,v) 为：
D(b,v)={\min\funcapply}\below{\mathrm{path\ }\gamma:b\rightsquigarrow v}\sum_{e\in\gamma}\ell_{\mathrm{eff}}(e)

其中每条边 e=(x,y) 的有效长度定义为：
\ell_{\mathrm{eff}}(e)=\parallel e\parallel\cdot\sqrt{{\hat{e}}^T\mathrm{\thinsp} M_{\mathrm{avg}}\mathrm{\thinsp} \hat{e}}

	\parallel e\parallel 为边的欧氏长度（或测地距离近似，通常直接用边长）
	\hat{e}=e/\parallel e\parallel 为边的单位方向向量（在全局坐标系中）
	M_{\mathrm{avg}}=\frac{M(x)+M(y)}{2} 为边两端点度量张量的平均值
该距离可通过 Dijkstra 算法在网格上高效计算（以有效长度作为边权）。
3. 几何属性一致性条件
为避免分区跨越几何特征剧烈变化的区域（如棱边、尖点），还需要检查候选顶点 v 与基准点 b 的几何属性差异是否在允许范围内。定义属性差异向量：
\Delta(v,b)=∣K(v)-K(b)∣σKarccos⁡(n(v)⋅n(b))σn∣erolled(v)-erolled(b)∣σR

其中 \sigma_K,\sigma_n,\sigma_R 为归一化因子（可取全局标准差）。则属性一致性条件为：
\parallel\Delta(v,b)\parallel_2\le\theta_{\mathrm{attr}}

其中 \theta_{\mathrm{attr}} 为预设阈值（例如 1.0~2.0）。该条件保证分区内的几何属性（曲率、法向、可展性偏差）与基准点保持相似。
4. 综合相似性判定
顶点 v 与基准点 b 属于同一分区当且仅当同时满足：
	距离条件：D(b,v)\le R_{\mathrm{max}}
其中 R_{\mathrm{max}} 为分区最大有效半径（全局超参数，控制分区大小）。
	属性条件：\parallel\Delta(v,b)\parallel_2\le\theta_{\mathrm{attr}}
这两个条件共同决定了分区的边界。距离条件决定了形状（由曲率引导），属性条件防止跨越几何突变区域。
5. 参数说明与泛化性
参数	含义	典型范围	对分区形状的影响
\alpha	曲率拉伸强度	0 ~ 10	越大，分区越倾向沿低曲率方向拉长
R_{\mathrm{max}}	最大有效半径	2 ~ 10 倍平均边长	控制分区总体大小
\theta_{\mathrm{attr}}	属性差异阈值	0.5 ~ 2.0	越小，分区内部几何越均匀
该方案不依赖人为指定的方向，完全由曲面自身的曲率张量驱动，因此对自由曲面具有泛化能力。对于您提出的三个测试案例：
	圆柱：环向曲率大 → 环向有效距离大 → 分区优先沿轴向生长 → 长条形矩形。
	圆锥：母线方向曲率小且方向渐变 → 分区沿母线方向生长，同时由于母线方向逐渐变化，分区宽度受限 → 自然形成三角形。
	无拉伸扭曲平面：曲率几乎为零 → 度量各向同性 → 分区近似圆形（由测地距离决定）。
6. 实现要点（供代码参考）
	预计算：每个顶点的 M(v) 可预先计算并存储为 3×3 对称矩阵（或仅存储两个主方向及曲率，实时计算）。
	Dijkstra 生长：从基准点开始，使用优先队列，边权为 \ell_{\mathrm{eff}}(e)，同时记录每个顶点的属性差异累积（但属性条件应独立判断，因为它是与基准点的直接比较，不是累积）。实际算法：
python
def grow_region(benchmark, mesh, alpha, R_max, theta_attr):
    dist = {v: inf}
    dist[benchmark] = 0
    queue = [(0, benchmark)]
    region = set()
    while queue:
        d, u = heapq.heappop(queue)
        if d > R_max: continue
        if norm_attr_diff(u, benchmark) > theta_attr: continue
        region.add(u)
        for v in neighbors(u):
            edge_len = effective_length(u, v, alpha)
            new_d = d + edge_len
            if new_d < dist[v]:
                dist[v] = new_d
                heapq.heappush(queue, (new_d, v))
    return region
	有效长度计算：对于边 (u,v)，取中点或平均度量 M_{avg}，计算 \sqrt{{\hat{e}}^TM_{avg}\hat{e}} 乘以欧氏边长。注意 \hat{e} 需在全局坐标系下，且度量张量定义在切平面上？实际上 M(v) 是定义在三维空间中的，但应限制在切平面内。为简化，可直接使用三维向量，因为法向分量会被 d_1,d_2 张成的平面吸收。更精确的做法是将边向量投影到切平面再计算，但通常网格边近似在曲面上，误差可接受。
7. 与后续模块的接口
相似性判定模块输出一个布尔值（是否属于同一分区），被区域生长模块调用。该模块的输入为：
	基准点 b（索引）
	候选顶点 v（索引）
	全局参数 \alpha,R_{\mathrm{max}},\theta_{\mathrm{attr}}
输出 True/False。
在区域生长过程中，为避免重复计算属性差异，可以在 Dijkstra 的每个节点第一次弹出时检查属性条件。注意属性条件是与基准点的直接比较，不随路径变化，因此可以提前计算所有顶点与基准点的属性差异，但那样效率低。更实际的做法是在生长过程中实时计算。
